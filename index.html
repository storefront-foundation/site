Meet React Storefront 

Build and deploy lightning-fast eCommerce websites in record time with a framework that is built to sell. 

Ludicrous Speed 

React Storefront let's you optimize every real and user-perceived performance metric to the max with server-side rendering, auto-AMP conversion, predictive prefetching, cache optimization, client data reuse, skeletons, and more.

Built on Next.js

React Storefront extends upon Next.js, so developers can just download and start coding. The framework also guides developers to performance best practices, so you know that your eCommerce PWA is being built for speed. 

Made to Sell

React Storefront is custom-built for eCommerce, including UI components and templates designed specifically for eCommerce. The framework is also search engine friendly, and allows you to preserve your existing URL scheme. 

Built for Complex Sites

React Storefront scales from $10M to $1B+ revenue sites. The framework supports real-world migration of complex eCommerce sites to PWAs in incremental steps.

React Storefront Features  

Everything you need to build ligtning-fast, converting eCommerce PWAs, quickly and easily.  

Universal Router 

Declare client and server routes and control caching from React Storefront's universal router. 

State Management 

React Storefront provides structured state management using mobx-state-tree.

Theming 

Easily tweak and control the look and feel of app components with custom themes and/or JSS.

Code splitting 

Bundle sizes are minimized by splitting page components into distinct bundles. 

 Integrate with GTM and  individual 3rd-party analytics in just a few of clicks. 

Analytics 

Includes multiple components for common eCommerce features built on top of Material UI. 

React Components 

Prefetching 

React Storefront makes it easy to prefetch and cache links as the user scrolls them into view.  

 Client & Edge Caching  

React Storefront provides a simple directive that gives you full control over browser caching.

Automatic Service Worker

The framework generates a service worker that supports static and runtime caching. 

SSR by Default 

React Storefront automatically renders the initial landing page on the server by default. This means faster load times for users and better SEO. No configuration is necessary.
